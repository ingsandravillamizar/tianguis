Directives

Es una herramienta que nos proporciona angular para dar comportamientos adicionales a nuestros elementos en el DOM (Document Object Model / Modelo del objeto de documento)

Tipos de directivas
  • Component directives: Estas son las directivas mas comunes y son los que definen plantillas, estilos y comportamiento para el DOM
  • Structural directives: Estas nos sirven para cambiar el diseño de nuestro DOM. Un ejemplo puede ser: *ngIf
  • Attribute Directives: Cambiar el estilo de un elemento. Ejemplo: NgClass

Mas populares

  • Structural Directives: *ngIf , *ngFor , * ngSwitch
  • Attribute Directives: NgClass y NgStyle

*ngIf

Se utiliza para definir si se renderiza o no en el template nuestro elemento


Su sintaxis es la siguiente en el elemento ponemos como atributo *ngIf="Condicion"
